# Parcial-1-Programacion-Avanzada
Sistema de Gestion de Garage

Este programa es el desarrollo del Parcial 1 para la materia Programacion Avanzada. Permite administrar el ingreso y salida de vehiculos en un estacionamiento, controlando el espacio y los costos.

Integrantes:Axel Brana y Matias Marone

Repositorio Git: https://github.com/axelbrana00/Parcial-1-Programacion-Avanzada
Link al codigo y los commits del desarrollo: https://github.com/axelbrana00/Parcial-1-Programacion-Avanzada/commits/main/

Funcionalidades del sistema:
- Registro de ingreso de Motos, Autos y Camiones.
- Control de capacidad maxima del garage y espacio por tipo de vehiculo.
- Registro de salida con calculo automatico de costo segun las horas.
- Validacion de patentes para que no se repitan.
- Manejo de errores para que el programa no se cierre si algo falla.

Como probar el programa:
1. Abrir el proyecto en Eclipse.
2. Ejecutar el archivo Main.java dentro del paquete Parcial1.
3. Seguir las instrucciones del menu en la consola.
